# Foccacia

Node.js web application developed as an academic project at ISEL.

## Overview

Foccacia is a JavaScript/Node.js project with an Express-based HTTP server, API/service modules and server-rendered views using Handlebars. The repository also includes project documentation and a report PDF.

The project focuses on building a web application with routing, data access, server-side rendering and integration with external/service modules.

## Main features

- Express HTTP server
- API and service-layer modules
