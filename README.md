# Foccacia

Node.js web application developed as an academic project at ISEL.

## Overview

Foccacia is a JavaScript/Node.js project with an Express-based HTTP server, API/service modules and server-rendered views using Handlebars.

## Main features

- Express HTTP server
- API and service-layer modules
- Handlebars views
- CORS support
- Session/authentication-related dependencies

## Tech stack

- JavaScript
- Node.js
- Express
- Handlebars
- Passport
